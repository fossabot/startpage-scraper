Get the top 10 search results from StartPage.com in your terminal.

## Develop Locally
* Clone the repository
* * `cd` into the cloned directory
* Install tools:
  * [Node.js](https://nodejs.org/en/)
  * [Yarn](https://yarnpkg.com/en/)
* Install dependencies:
  * `yarn`
* Start
  * `yarn start`
